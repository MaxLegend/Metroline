# Metroline



**Приложение работает как редактор схем метрополитена с возможностью создания станций различных типов, соединения их туннелями, добавления рек и полной настройки визуального представления карты метро.**

**Приложение работает на JVM - не забудьте установить, если ее у вас нет: https://www.oracle.com/java/technologies/downloads/**

### Основные Возможности

**1. Управление станциями:**
- Создание/удаление станций метро
- 14 типов станций: Regular, Transfer, Terminal, Transit, Planned, Building, Closed, Destroyed, Abandoned, Drowned, Ruined, Repair, Depo, Burned
- Назначение цвета линии станции
- Операции с названиями станций (перемещение вокруг, переименования)
- Создание связей между станциями в октагональном пространстве
- Custom properties для станций

**2. Управление туннелями:**
- Соединение станций туннелями
- 4 типа туннелей: Active, Planned, Building, Destroyed
- Настройка цвета туннеля
- Поддержка умных туннелей через PathPoint
- Автоматический расчёт пути 

**3. Управление реками:**
- Создание рек через систему точек 
- Настройка цвета и ширины реки
- Интерполированный рендеринг пути

**4. Работа с миром:**
- Создание игрового мира заданного размера (GameWorld)
- Тайловая система 
- Настройка цвета мира
- Сохранение/загрузка проектов 

**5. UI и Управление:**
- Мышь: создание/выбор объектов 
- Клавиатура: обширная сетка горячих клавиш и сочетаний
- Система выделения объектов 
- Popup-меню для типов станций/туннелей
- Легенда линий с возможность редактирования названия линий
- Окно информации об станции со свойствами

**6. Дополнительные функции:**
- Логирование действий (MetroLogger)
- Локализация интерфейса (LngUtil)
- Глобальные настройки приложения
- Crash reporting система
- Работа в оконном режиме (максимизированное окно)

### Архитектура
```
Core: GameObject → Station/Tunnel/River
World: World → GameWorld (tile system)
UI: MainFrame → Screens (Menu, Game, Settings)
Render: StationRender, TunnelRender, RiverRender
Control: Mouse/Keyboard/Click Controllers
Utils: Serializer, Logger, Color/Image utilities
```
